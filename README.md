# QA Portfolio — SauceDemo

## О проекте

Учебный проект по тестированию веб-приложения интернет-магазина SauceDemo.

Цель проекта — продемонстрировать навыки ручного тестирования, включая анализ функциональности, написание тестовой документации и оформление баг-репортов.

---

## Объект тестирования

- Веб-приложение: https://www.saucedemo.com/
- Тип: интернет-магазин

---

## Область тестирования

В рамках проекта протестированы основные пользовательские сценарии:

- Авторизация пользователя
- Работа с каталогом товаров
- Добавление и удаление товаров из корзины
- Оформление заказа (checkout)
- Подтверждение заказа

---

## Используемые инструменты

- DevTools (Chrome)
- Qase (тест-кейсы)
- Git / GitHub
