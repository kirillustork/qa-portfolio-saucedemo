## BUG-001

**Заголовок:**  
При отправке формы с пустыми полями отображается сообщение только о Username

**Предусловия:**  
Открыта страница авторизации https://www.saucedemo.com/

**Шаги:**

1. Оставить поля Username и Password пустыми
2. Нажать кнопку Login

**Фактический результат:**  
Отображается сообщение об ошибке "Epic sadface: Username is required"

**Ожидаемый результат:**  
Отображается сообщение об ошибке для всех незаполненных обязательных полей

**Severity:** Low  
**Priority:** Low

## BUG-003

**Заголовок:**  
Возможно продолжить оформление заказа с невалидными данными в форме "Your information"

**Предусловия:**  
Пользователь авторизован в системе, открыта страница https://www.saucedemo.com/checkout-step-one.html

**Шаги:**

1. Ввести в поле First Name значение "123"
2. Ввести в поле Last Name значение "!!!"
3. Ввести в поле Zip Code значение "abc"
4. Нажать кнопку Continue

**Фактический результат:**  
Происходит переход на страницу checkout-step-two (Overview)

**Ожидаемый результат:**  
Отображается сообщение об ошибке, переход на следующий шаг не выполняется

**Severity:** Medium  
**Priority:** Medium

## BUG-004

**Заголовок:**  
Кнопка Cancel на странице Overview ведёт на страницу каталога вместо предыдущего шага оформления заказа

**Предусловия:**  
Пользователь авторизован в системе, открыта страница https://www.saucedemo.com/checkout-step-two.html

**Шаги:**

1. Нажать кнопку Cancel

**Severity:** Low  
**Priority:** Medium
